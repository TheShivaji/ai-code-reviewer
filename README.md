## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** v18+
- **Gemini API Key** вҶ’ [Get it free at Google AI Studio](https://aistudio.google.com/)

### Installation

```bash
# Clone the repo
git clone https://github.com/TheShivaji/ai-code-reviewer.git
cd ai-code-reviewer

# Backend setup
cd Backend
npm install

# Configure environment
cp .env.example .env
```

```env
# .env
GEMINI_API_KEY=your_gemini_api_key_here
PORT=3000
```

```bash
# Start the server
npm start

# Open Frontend вҖ” just open in browser
cd ../Frontend
open index.html    # or use VS Code Live Server
```

> Server runs at `http://localhost:3000`
